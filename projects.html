Tic Track

Tic Track is a simple management app for your tasks. It allows you to create, edit, and delete tasks easily. You can create marks to categorize your tasks and filter them by status. I built it with Flutter, Dart and Hive for the storage. Unfortunately, this app is only available on MacOS for now.

Louvain Market

Louvain Market is a web application that allows students to buy and sell items. It features user authentication, item listings, a rating for every users and a admin panel for managing items and users. We built it as a group project using HTML, Tailwind CSS, JavaScript, Node JS, Express and for the database we used MongoDB.

Page Rank Algorithm

This project implements the Page Rank algorithm, which is used by Google to rank web pages in their search results. We built it as a group project for our LINFO1114 course. The algorithm is implemented in Python.

K-Means Algorithm

This project implements the K-Means algorithm, which is a popular clustering algorithm used in machine learning or image compression. I built it using C.

Eggcellent Timer

Eggcellent Timer is a mobile app designed to help you cook the perfect egg every time. Whether you prefer soft-boiled, medium-boiled, or hard-boiled eggs, this app provides timers tailored to each type. Built with Flutter and Dart, it ensures a user-friendly experience on both iOS and Android.

MobiliWeb Website

MobiliWeb is a web application that displays a lot of data about transport in the various cities and streets in the dataset. You can also filter by time period. We built it as a group project with HTML, CSS, JavaScript, Flask and for the database part we used MySQL.
